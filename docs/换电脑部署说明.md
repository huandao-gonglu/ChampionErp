# 换电脑部署说明

生成时间：2026-06-01

## 1. 新电脑需要安装

必须安装：

1. Python 3.10 或以上。
2. Google Chrome。
3. Git，建议安装。
4. Playwright 依赖，如果要运行原项目端到端测试或浏览器采集增强。

建议安装：

1. VS Code。
2. VS Code Python 插件。
3. VS Code SQLite Viewer 插件。
4. VS Code GitLens 插件。
5. PowerShell 7，可选。

## 2. Python 版本要求

建议 Python 3.10 或以上。当前 Web 底座使用标准库 `ThreadingHTTPServer`，并依赖项目中的 Python 包。

确认命令：

```powershell
python --version
```

如果系统没有全局 Python，也可以使用项目 `.venv\Scripts\python.exe`。

## 3. pip 依赖

当前 `requirements.txt` 包含：

```text
openai>=1.0.0
pillow>=10.0.0
```

安装命令：

```powershell
cd /d D:\champion-Erp
python -m pip install -r requirements.txt
```

如果后续启用 Playwright Python 采集：

```powershell
python -m pip install playwright
python -m playwright install
```

当前项目还包含 Node Playwright 测试依赖，若运行 `package.json` 中的测试：

```powershell
cd /d D:\champion-Erp
npm install
npx playwright install
npm run test:e2e
```

## 4. 是否需要 Chrome

需要。Chrome 用于：

1. 打开本地 ERP 页面。
2. 1688 / Amazon 等需要登录或验证码的浏览器会话采集。
3. Remote Debugging 浏览器模式。

默认 Chrome 路径通常是：

```text
C:\Program Files\Google\Chrome\Application\chrome.exe
```

## 5. 是否需要 Playwright

日常打开 Web ERP 不强制需要 Playwright。

以下场景需要：

1. 运行原项目 Playwright 端到端测试。
2. 启用 `ERP_USE_PLAYWRIGHT=1` 的浏览器采集路径。
3. 自动化浏览器检查页面。

## 6. API Key 在哪里配置

当前项目会读取本地配置文件，例如：

`D:\champion-Erp\app_config.json`

以及店铺/平台配置文件，例如：

`D:\champion-Erp\store_config.json`

换电脑时建议重新检查：

1. OpenAI API Key。
2. Mercado Libre client id / client secret。
3. Mercado Libre access token / refresh token。
4. 各平台站点、类目、价格、佣金、物流配置。

注意：不要把包含密钥的配置文件发给不可信的人，也不要提交到公开仓库。

## 7. Mercado Libre 是否需要重新授权

建议重新授权或至少重新校验 token。

原因：

1. `access_token` 可能过期。
2. `refresh_token` 可能失效或被平台轮换。
3. 新电脑环境、浏览器 profile、回调地址和账号登录状态可能不同。

第一阶段不要接真实发布。第二阶段只做 payload 预览和预检。真实发布前必须重新确认授权状态。

## 8. 1688 / ChatGPT 浏览器 profile 是否需要重新登录

需要。

浏览器 profile 通常绑定本机路径、Cookie、登录状态和安全验证。换电脑后建议：

1. 在新电脑启动专用 Chrome profile。
2. 手动登录 1688。
3. 手动完成滑块、验证码或安全验证。
4. 如需 ChatGPT 浏览器流程，也在新电脑重新登录。

不要直接依赖旧电脑的浏览器 profile。

## 9. 可以跟随 D:\champion-Erp 直接拷贝的数据

可以拷贝：

1. `erp_web_app.py`
2. `erp_web_template.html`
3. `erp_db.py`
4. `product_model.py`
5. `publishing_bus.py`
6. `services`
7. `docs`
8. `tests`
9. `erp.sqlite3`
10. `output` 下的非敏感图片、日志、缓存文件
11. 类目缓存
12. 发布记录文件，但需要确认是否包含 token 或敏感 payload

## 10. 不能直接拷贝或需要重新配置的数据

建议新电脑重新配置：

1. Mercado Libre 授权 token。
2. OpenAI API Key。
3. 1688 登录状态。
4. ChatGPT 登录状态。
5. Chrome remote debugging profile。
6. 本机绝对路径配置。
7. 任何包含账号、密钥、cookie、token 的文件。

## 11. 启动方式

双击：

```text
D:\champion-Erp\打开网页版ERP.cmd
```

默认访问地址：

```text
http://127.0.0.1:5000/
```

第一阶段保持 5000 端口，不改 8765。

## 12. 换电脑后的检查清单

1. `D:\champion-Erp\打开网页版ERP.cmd` 能启动。
2. `http://127.0.0.1:5000/` 能打开。
3. SQLite 能读取 `products`, `media_assets`, `platform_drafts`, `publish_logs`。
4. 商品编辑页面能显示。
5. 图片池能显示；如果历史图片不显示，先检查是否仍指向旧电脑绝对路径。
6. 发布预检能打开。
7. 发布日志能打开。
8. 不进行真实发布，直到 Mercado Libre 授权和 payload 预览全部确认。
